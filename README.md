# SciJava Incubator: A staging ground for fledgling SciJava projects

This project is intended to be the starting point for the majority of SciJava projects. It serves as a place for fledgling projects to grow unhindered by versioning, packaging and project restrictions.

**Note: It is strongly discouraged for non-incubator projects to depend on incubator projects**

# What benefits come with using the SciJava Incubator?:
* Leading edge: Take advantage of Java 11, the [Java Platform Module System](https://en.wikipedia.org/wiki/Java_Platform_Module_System), and other fledgling projects without waiting for them to arrive in established projects.
* Flexibility: With your project in the incubator, it is easy to partition a project into multiple modules. There is no need to create new repositories, as you don't have to create any until your project is ready to graduate.
